# Tic-Tac-Toe (CLI)

A simple command-line **Tic-Tac-Toe** game built with Python for two players. Players take turns placing **X** and **O** until one wins or the game ends in a draw.

## Features

- Two-player gameplay
- Command-line interface
- Win detection
- Draw detection
- Input validation
- Prevents overwriting occupied cells

## Run the Project

```bash
python Tic_Tac_Toe.py
```

## How to Play

The board positions are numbered as follows:

```
1 | 2 | 3
--+---+--
4 | 5 | 6
--+---+--
7 | 8 | 9
```

- Player **X** goes first.
- Enter the position number (1-9).
- First player to align **3 symbols** horizontally, vertically, or diagonally wins.
- If all spaces are filled without a winner, the game ends in a draw.
