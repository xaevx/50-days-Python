# Password Manager

A simple command-line **Password Manager** built with Python that allows users to securely store, generate, search, and delete passwords for different accounts.

## Features

- Add New Password
- Generate Strong Random Passwords
- View Saved Passwords
- Search Password by Account
- Delete Saved Password
- Simple CLI Interface
- Input Validation

## Run the Project

```bash
python Password_Manager.py
```

## Modules Used

- Python 3
- `random`
- `string`

## What i learned

- Dictionaries
- Functions
- Loops
- Conditional Statements
- User Input
- Random Module
- String Module
- Exception Handling

> **Note:** Passwords are stored only in memory while the program is running. They are **not saved permanently** after closing the application.
