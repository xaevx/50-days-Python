# Contact Book

A simple command-line **Contact Book** built with Python that allows you to store and manage contacts.

## Features

- Add Contact
- View All Contacts
- Search Contact
- Delete Contact
- Simple CLI Interface
- Input Validation

## Run the Project

```bash
python Contact_Book.py
```

## Modules Used

- Python 3

## What i learned

- Dictionaries
- Functions
- Loops
- Conditional Statements
- User Input
- Exception Handling
