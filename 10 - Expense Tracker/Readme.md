# Expense Tracker

A simple command-line **Expense Tracker** built with Python to record, manage, and track daily expenses.

## Features

- Add Expense
- View All Expenses
- Delete Expense
- Calculate Total Expense
- Simple CLI Interface
- Input Validation

## Run the Project

```bash
python Expense_Tracker.py
```

## Modules Used

- Python 3

## What i learned

- Lists
- Dictionaries
- Functions
- Loops
- Conditional Statements
- User Input
- Exception Handling
