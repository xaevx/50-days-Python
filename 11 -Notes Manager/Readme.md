# Notes Manager

A simple command-line **Notes Manager** built with Python that allows users to create, view, and delete notes.

## Features

- Add Notes
- View All Notes
- Delete Notes
- Clear All Notes
- Simple CLI Interface
- Input Validation

## Run the Project

```bash
python Notes_Manager.py
```

## Modules Used

- Python 3

## What i learned

- Lists
- Functions
- Loops
- Conditional Statements
- User Input
- Exception Handling
