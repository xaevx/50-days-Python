# Password Generator

A simple command-line Password Generator built with Python that generates secure and customizable passwords.

## Features

- Custom password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special symbols
- Multiple password generation
- Input validation

## Run the Project

```bash
python password_generator.py
```

## Modules Used

- Python 3
- `random`
- `string`
