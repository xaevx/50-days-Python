# Library Management System

A simple command-line **Library Management System** built with Python to manage books in a library. Users can add, view, issue, return, and remove books through an easy-to-use menu.

## Features

- Add New Books
- View Available Books
- Issue Books
- Return Books
- Remove Books
- Simple CLI Interface
- Input Validation

## Run the Project

```bash
python Library_Management.py
```

## Modules Used

- Python 3

## What i learned

- Dictionaries
- Nested Dictionaries
- Functions
- Loops
- Conditional Statements
- User Input
- Exception Handling

> **Note:** The library data is stored only in memory. All records will be lost once the program is closed.
