# Number Guessing Game

A simple command-line Number Guessing Game built with Python. The program generates a random number between **1 and 100**, and the player must guess it with hints provided after each attempt.

## Features

- Random number generation
- Hint system (Too High / Too Low)
- Tracks the number of attempts
- Play again option
- Input validation using exception handling

## Run the Project

```bash
python Number_Guessing_Game.py
```

## How to Play

1. The program generates a random number between **1** and **100**.
2. Enter your guess.
3. The program will tell you if your guess is:
   - Too Low
   - Too High
4. Keep guessing until you find the correct number.
5. Choose whether to play another round.

- Python 3
- `random` module
