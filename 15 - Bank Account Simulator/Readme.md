# Bank Account Simulator

A simple command-line **Bank Account Simulator** built with Python using **Object-Oriented Programming (OOP)**. The application allows users to perform basic banking operations such as deposits, withdrawals, checking balances, and viewing account details.

## Features

- Deposit Money
- Withdraw Money
- Check Account Balance
- View Account Details
- Input Validation
- Built using Python Classes and Objects

## Run the Project

```bash
python Bank_Account_Simulator.py
```

## Modules Used

- Python 3

## What i learned

- Classes & Objects
- Constructors (`__init__`)
- Object Methods
- Encapsulation
- Conditional Statements
- Exception Handling
- User Input
- Loops

> **Note:** This is a simulation project. Account information exists only while the program is running and is not stored permanently.
