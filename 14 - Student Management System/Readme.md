# Student Management System

A simple command-line **Student Management System** built with Python to manage student records. The application allows users to add, view, search, update, and delete student information.

## Features

- Add Student Records
- View All Students
- Search Student by Roll Number
- Update Student Marks
- Delete Student Records
- Simple CLI Interface
- Input Validation

## Run the Project

```bash
python Student_Management.py
```

## Modules Used

- Python 3

## What i learned

- Dictionaries
- Nested Dictionaries
- Functions
- Loops
- Conditional Statements
- User Input
- Exception Handling
- CRUD Operations

> **Note:** Student records are stored only in memory. Data will be lost when the program is closed.
