# Inventory Management System

A simple command-line **Inventory Management System** built with Python using **Object-Oriented Programming (OOP)**. It allows users to manage inventory items by adding, updating, viewing, and deleting products.

## Features

- Add New Items
- View Inventory
- Update Item Details
- Delete Items
- Calculate Total Inventory Value
- Input Validation
- Built using Classes and Objects

## Run the Project

```bash
python Inventory_Management.py
```

## Modules Used

- Python 3

## What i learned

- Object-Oriented Programming (OOP)
- Classes & Objects
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Exception Handling
- CRUD Operations

> **Note:** Inventory data is stored only in memory and will be lost when the program is closed.
