# Smart Python Calculator

A simple command-line calculator built with Python that performs basic and advanced mathematical operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Power
- Modulus
- Floor Division
- Square Root
- Percentage Calculator
- Calculation History
- Error Handling

## Run the Project

```bash
python Calculator.py
```

## Module used

- Python 3
- Math Module

## What I Learned

- Functions
- Loops
- Conditional Statements
- Exception Handling
- Lists & Dictionaries
- User Input Handling
