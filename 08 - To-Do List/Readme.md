# To-Do List

A simple command-line **To-Do List** application built with Python. It helps you manage your daily tasks by allowing you to add, view, remove, and clear tasks.

## Features

- Add new tasks
- View all tasks
- Remove a task
- Clear all tasks
- Simple command-line interface
- Input validation

## Run the Project

```bash
python Todo_List.py
```

## Modules Used

- Python 3

- Functions
- Loops
- Conditional Statements
- User Input
