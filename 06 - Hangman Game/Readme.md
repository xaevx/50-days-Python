# Hangman Game

A simple command-line **Hangman Game** built with Python. Guess the hidden word one letter at a time before you run out of attempts.

## Features

- Random word selection
- Letter-by-letter guessing
- Displays hidden letters with `_`
- Tracks remaining attempts
- Prevents duplicate guesses
- Input validation
- Win/Lose detection

## Run the Project

```bash
python Hangman_Game.py
```

## How to Play

1. A random word is selected.
2. Guess one letter at a time.
3. If the letter exists in the word, it is revealed.
4. Incorrect guesses reduce your remaining attempts.
5. Guess the entire word before your attempts reach **0**.
