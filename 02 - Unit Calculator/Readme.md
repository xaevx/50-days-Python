# Unit Converter

A simple command-line Unit Converter built with Python that converts values between different units of measurement.

## Features

- Length Converter
- Weight Converter
- Temperature Converter
- User-friendly menu
- Input validation
- Error handling

## Run the Project

```bash
python unit_converter.py
```

## Supported Units

### Length

- mm
- cm
- m
- km

### Weight

- mg
- g
- kg

### Temperature

- C (Celsius)
- F (Fahrenheit)
- K (Kelvin)

## Built With

- Python 3
