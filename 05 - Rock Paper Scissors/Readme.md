# Rock Paper Scissors

A simple command-line **Rock Paper Scissors** game built with Python. Play against the computer and keep track of your score.

## Features

- Play against the computer
- Random computer choices
- Score tracking
- Play unlimited rounds
- Input validation
- Simple command-line interface

## Run the Project

```bash
python Rock_Paper_Scissor.py
```

## How to Play

1. Choose:
   - Rock
   - Paper
   - Scissors
2. The computer randomly selects its move.
3. The winner is decided using the classic rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Scores are updated after each round.
5. Enter **0** to exit the game.
